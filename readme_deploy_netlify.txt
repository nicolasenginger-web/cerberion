==============================
DEPLOIEMENT SUR NETLIFY (2 MIN)
==============================
1) Aller sur https://app.netlify.com et se connecter (Google ou email).
2) Cliquer sur "Add new site" > "Deploy manually".
3) Faire glisser le ZIP "site_cerberion_presentation.zip" dans la fenêtre.
4) Netlify génère une URL publique du type : https://xxxxx.netlify.app
5) Copier ce lien et l'envoyer par email / WhatsApp.

OPTION : NOM DE DOMAINE PERSONNALISE
- Dans "Site configuration" > "Domain management" > "Add custom domain".
- Ajouter par ex. presentation.cerberion.lu
- Créer un enregistrement CNAME chez votre registrar pointant vers le sous-domaine Netlify.
- Activer le HTTPS (Let's Encrypt) en 1 clic.

MISES A JOUR ULTRA SIMPLES
- Pour modifier le texte ou les liens, ouvrez index.html, enregistrez, et redéployez en déposant le nouveau fichier (ou ZIP).
